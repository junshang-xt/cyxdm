# 创意小代码 - 按钮网格项目

## 项目介绍

这是一个具有现代UI设计的按钮网格项目，每个按钮都有独特的动画效果和交互体验。项目包含以下主要功能：

- 36个带有水波纹动画效果的按钮
- 响应式网格布局设计
- 点击效果和粒子动画
- 键盘导航支持
- 每个按钮点击后跳转到专属页面

## 页面跳转功能

### 已实现的功能

1. 点击按钮后，会触发平滑的动画效果
2. 显示"正在跳转到页面X..."的提示信息
3. 跳转到`pages/pageX.html`（其中X为按钮编号）
4. 目标页面包含统一的设计风格和返回首页功能

### 已创建的专属页面

目前已创建了以下示例页面：
- `pages/page1.html` - 创新设计模式
- `pages/page2.html` - 前沿技术探索

### 如何创建更多页面

您可以按照以下步骤为剩余按钮创建专属页面：

1. 复制现有的页面模板（如`page1.html`或`page2.html`）
2. 重命名为`pageX.html`，其中X为按钮编号（3-36）
3. 修改页面内容，包括：
   - 页面标题和副标题
   - 核心内容描述
   - 功能列表
   - 渐变色（建议使用不同的颜色组合以区分页面）

### 页面模板内容说明

每个页面包含以下元素：

- **页面标题** - 显示页面编号和动态渐变效果
- **内容区域** - 包含主题标题和详细描述
- **功能列表** - 展示核心功能点
- **返回按钮** - 点击可返回首页
- **共享样式** - 与主页保持一致的设计风格

## 技术栈

- HTML5
- CSS3 (动画、渐变、响应式设计)
- JavaScript (交互效果、动画控制)

## 使用说明

1. 在浏览器中打开`index.html`
2. 使用鼠标点击按钮或键盘导航（方向键、Tab键）
3. 点击按钮后会跳转到对应的专属页面
4. 在专属页面中，点击"返回首页"按钮可返回主页面

## 自定义建议

- 您可以修改`styles.css`中的动画参数，调整水波纹效果
- 可以更改按钮的颜色、渐变和悬停效果
- 为专属页面添加更多定制内容，如图片、示例代码等

## 注意事项

- 确保所有页面文件路径正确，特别是CSS文件的引用路径
- 如果在本地文件系统中直接打开页面，某些浏览器可能限制跨域资源访问
- 对于未创建的页面，点击后会显示跳转提示但可能无法加载页面内容
